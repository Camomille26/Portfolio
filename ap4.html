AP4: Conception d'une application web

Objectif

Aux côtés de trois autres camarades de projet, nous devons réaliser une application mobile via AndroidStudio. Cette application codé en Java doit permettre aux clients utilisant l'application de se connecter avec son compte client, de pouvoir scanner les QR code des articles afin de voir apparaître leur descriptions et également pouvoir utiliser la géolocalisation afin que l'application puisse faire apparaître le magasin ALL4SPORT le plus proche de l'utilisateur. Il nous faudra également réaliser un webservice à partir du code source de votre projet web AP3 afin de récupérer les informations envoyées par l’application mobile et mettre à jour la liste des commandes du magasin.
